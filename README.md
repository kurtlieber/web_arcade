# 🚀 SPACE INVADERS (1978 Recreation)

An HTML5 recreation of the original **1978 Taito Space Invaders** arcade game.

## Features
- 55 invaders (5×11) with 3 types + 2-frame animation
- 4 destructible bunkers
- Mystery Ship (UFO) with cycling 50/100/150/300 scoring
- Classic 4-note marching tune that speeds up
- Authentic speed increase as invaders die
- One bullet at a time (player)
- Extra life at 1500 points
- Game over if invaders reach bottom

## Controls
- **Arrow keys** / **A D** — Move
- **Space** — Shoot
- **P** — Pause
- **R** — Restart (after game over)

## How to Play Locally
Just open `index.html` in any modern browser.

## Deployment
Hosted on **Cloudflare Pages** + GitHub for automatic deployments.
